## ⚙️ Instalación y Ejecución
<details open>
<summary><b>Guía Paso a Paso</b></summary>

```bash
# 1. Clonar repositorio
git clone https://github.com/tu-usuario/recumet-bolivia.git
cd recumet-bolivia

# 2. Instalar dependencias
npm install
# o
pnpm install
# o
yarn install

# 3. Iniciar el servidor de desarrollo
npm start
